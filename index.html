form - foydalanuvchi tomonidan input, textarea va select kab teg lar orqali kiritilgan va tanlangan ma'lumotlarni serverga yuboruvchi teg
action - server manzili
input - ma'lumot kiritish, to'ldirish uchun maydon
name - ma'lumot serverga borishi va qanday ma'lumot yuborilganligini bildirib turishi uchun
type - input turi
placeholder - input ichidagi yo'nalish beruvchi matn
<form action="https://echo.htmlacademy.ru">
    <input type="text" placeholder="ism-familiya" name="fullname">
    <button>Submit button</button>
</form>
label - ma'lumot beruvchi input ga bog'langan matn
for - input ning id sini yozish va input ga ulash uchun
id - takrorlanmas class ga o'xshash atribyut
<form action="https://echo.htmlacademy.ru">
    <label for="nameinput">Ism va familiya:</label>
    <input id="nameinput" type="text" placeholder="ism-familiya" name="fullname">
    <button>Submit button</button>
</form>
